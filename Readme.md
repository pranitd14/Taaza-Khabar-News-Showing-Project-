# Taaza Khabar (News Showing Project)

**Taaza Khabar** is a modern and responsive news website built using HTML, CSS, JavaScript, PHP, and MySQL. It provides users with a clean and interactive interface to read news while offering essential features like user authentication and theme toggling.

---

## 🚀 Features

- ✅ User Login & Registration (PHP + MySQL)
- 🌓 Dark/Light Mode Toggle
- 👁️ Show/Hide Password Functionality
- 🔁 One-Page Toggle Between Login & Register
- 🚨 SweetAlert for Success/Error Alerts
- 🖼️ Custom Logo Integration
- 💻 Responsive UI for all devices

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: PHP  
- **Database**: MySQL  
- **Server Environment**: XAMPP  
- **Libraries Used**: SweetAlert
